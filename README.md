# Source Code For FOCI
## Paper: "FOCI: Fair Cross-Network Node Classification via Optimal Transport"

## Setup
Ensure that you have all requirements installed. See `requirements.txt`


## Running the Code
You can run FOCI on the Pokec and compas datasets used in the paper by running the `run.py` script and indicating source and target datasets as shown below.

```bash
python3 run.py -s pokec_n -t pokec_z
```
